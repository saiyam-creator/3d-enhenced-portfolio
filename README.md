# 🚀 Saiyam Jain — AI Developer Portfolio

> A static but highly interactive 3D portfolio website with green screen videos, Three.js animations, AI chatbot, and gamified UI.

## ✨ Features

- **🎨 Immersive 3D Background** — Floating geometric shapes with dynamic lighting (Three.js)
- **🟢 Green Screen Video Characters** — Real-time chroma key removal (client-side) for hero, skills, and location sections
- **🤖 AI Chatbot (SaiBot)** — Rule-based assistant that answers questions about projects, skills, and contact
- **🏆 Achievements Modal** — Trophy button triggers a dragon green screen video, then opens certificates & badges modal with tabs
- **📁 Projects Gallery** — 15+ projects with category filters, favorite system (localStorage), and detailed modals
- **🎬 Video Demos** — Embedded Cloudinary walkthroughs of key projects
- **✨ Cool Effects** — Custom cursor, mouse trail, scroll progress bar, click particles, matrix code rain, floating orbs, section fade-in
- **📱 Fully Responsive** — Works on desktop, tablet, and mobile

## 🛠️ Tech Stack

| Category       | Technologies |
|----------------|--------------|
| Frontend       | HTML5, CSS3, JavaScript (ES6) |
| 3D Graphics    | Three.js |
| Icons & Fonts  | Font Awesome, Google Fonts (Bebas Neue, Space Grotesk, Fira Code) |
| Animations     | CSS keyframes, Intersection Observer, requestAnimationFrame |
| Data Storage   | localStorage (favorites) |
| Deployment     | GitHub Pages / Vercel / Netlify |
